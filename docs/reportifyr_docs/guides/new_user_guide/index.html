reportifyr New User Guide

Welcome to the reportifyr New User Guide! As reportifyr is in active, early-stage development, be sure to visit our GitHub page frequently to stay up-to-date with the latest features and releases.

What is reportifyr?

At its core, reportifyr is an R package built with analysts and authors in mind. It simplifies the creation of traceable, consistent Microsoft Word documents (reports) by standardizing analysis outputs (artifacts) and improving reproducibility through metadata management.
