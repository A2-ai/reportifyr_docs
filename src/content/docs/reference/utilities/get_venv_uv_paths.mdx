The `get_venv_uv_paths()` function in the `reportifyr` package locates the `Python` virtual environment (`.venv`) and the `uv` executable.

---

## `get_venv_uv_paths()`

<EmbeddedHTML filename="/public/html_files/get_venv_uv_paths.html"/>

---

### Returns

- **List**: A list of file paths to `.venv` subdirectory and the `uv` executable.

---

### Logging

- **Info**: Logs if `options('venv_dir')` is set to project root.
- **Error**: Logs if `.venv` or `uv` is not found.
