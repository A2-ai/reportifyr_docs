The `validate_config()` function in the `reportifyr` package reads a `config.yaml` file, checks every supported option for the correct type and, where relevant, an acceptable value, before returning a single Boolean indicating whether the configuration is valid.

---

## `validate_config()`

<EmbeddedHTML filename="/public/html_files/validate_config.html"/>

---

### Returns

- **Logical**: `TRUE` if all validation checks pass, otherwise `FALSE`.

---

### Logging

- **Debug**: Logs the start and end of the function, the file path to the `config.yaml`, and each attempt to check a supported option in the `config.yaml`.
- **Error**: Logs errors if the supported options in the `config.yaml` are of an incorrect type.
